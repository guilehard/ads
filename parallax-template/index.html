HTML5

HTML5 é a quinta e última versão do HTML, sigla de Hypertext Markup Language, uma das primeiras e principais linguagens de marcação da web. O HTML, serve basicamente para o desenvolvimento de páginas que contenham conteúdos simples, como textos, imagens, áudio, vídeos e etc.



Estrutura básica HTML

Um documento HTML recebe algumas tags que formam a sua estrutura básica.

No HTML5, o documento padrão recebe a seguinte estrutura:

<!DOCTYPE html> Indica que está sendo usado o HTML 5
<html> Início do código HTML
<head> providencia informações gerais (metadados) sobre o documento, incluindo seu título e links para scripts e folhas de estilos.
<title>Título da página</title> Mostra o título do site, na aba do navegador
<meta charset="utf-8"/> é uma tag meta utilizada para comunicar aos navegadores qual é o formato de codificação de caracteres
</head>
<body> O corpo do site, onde fica tudo que aparece em um site
</body>
</html>
</> Indica o fechamento da TAG



CSS

CSS (Cascading Style Sheet, ou Folha de Estilo em Cascatas), é usado para a estilização das linguagens de marcação (site). O CSS é responsavel por deixar um site com "Estilo", alterando cores, tamanhos de elementos, efeitos...



Estrutura básica CSS

As folhas de estilo funcionam basicamente através de regras que são vinculadas às tags HTML.

E o que torna fácil a criação de folhas de estilo é que essas regras obedecem a uma estrutura muito simples, composta por seletor, propriedade e valor.

p {
color: red;
}

O CSS usado na forma acima, faz com que: Um parágrafo fique com a cor vermelha.



Javascript

JavaScript é uma linguagem de programação que permite a você implementar itens complexos em páginas web — toda vez que uma página da web faz mais do que simplesmente mostrar a você informação estática — mostrando conteúdo que se atualiza em um intervalo de tempo, mapas interativos ou gráficos 2D/3D animados, etc.



Sintaxe JavaScript

A sintaxe basica do javascript pode ser vista logo abaixo

Nesse exemplo criamos um botão que muda a cor.

<element>elemento onclick="funçãoQueSeráExecutada()">Clique aqui</element>

Observe que o atributo onclick é puramente JavaScript. O valor que ele recebe, que é a função que será executada, diz tudo, já que ela é invocada diretamente da tag de abertura.
